    11ty SASS(SCSS) 사용

    기존 사이트의 테마를 옮기다 SCSS가 적용된 것을 보고 찾아보았다.

    간단히 말해 CSS를 전처리 하기 위한 스크립트 언어이다.

    SASS나 SCSS 구문은 브라우저가 직접 처리할 수 없고 작성한 파일을 컴파일 하여 CSS 형태로 출력해서 사용한다.

    SASS나 SCSS를 같이 묶어서 취급하는듯 하고 개인적으로는 중괄호가 있는 SCSS가 더 눈에 잘 들어온다.

    변수, 중첩, 루프 등은 둘째 치고 import가 있어서 부분부분 쪼개서 만들면 컴파일 시 하나의 파일로 출력되어 파일 하나만 link 해서 사용하면 되는 점이 편리하다.

    위키백과, 공식 사이트에 정리가 잘 되어있다.

    Eleventy와 함께 SASS 사용 문서를 보고 가볍게 해보았다.

    npm으로 설치한다.

    npm install sass --save-dev

    _sass 디렉토리에 main.scss를 만들었다.

    _sass 디렉토리의 내용을 컴파일하여 _site/assets/css 디렉토리에 출력한다.

    npx sass _sass:_site/assets/css

    원하는 곳에 연결하여 사용한다.

    <link rel="stylesheet" href="/assets/css/main.css">
