    일레븐티(11ty) 구문 강조

    jekyll에서의 구문강조 기능은 rouge 라는 것을 사용한다. 대문에 ~written in pure Ruby. 라고 써있고 js로의 포트는 없어보인다.

    11ty 공식 문서에 플러그인으로 Syntax Highlighting 항목이 있어 가볍게 사용해보았다.

    설치

    npm install @11ty/eleventy-plugin-syntaxhighlight --save-dev

    .eleventy.js 에 플러그인 추가 항목을 작성한다.

    const syntaxHighlight = require("@11ty/eleventy-plugin-syntaxhighlight");
    function eleventy(config) {
        // ...
        config.addPlugin(syntaxHighlight);
    }
    module.exports = eleventy;

    적용이 되긴 하는데 하얗게 나온다. CSS가 없어서 그렇다.

    CSS를 템플릿 등 원하는 곳에 연결한다.

    <link href="https://unpkg.com/prismjs@1.20.0/themes/prism-okaidia.css" rel="stylesheet">

    PrismJS 라이브러리를 사용하며 테마 기능도 있다고 한다.
