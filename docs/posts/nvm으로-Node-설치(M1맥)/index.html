    nvm으로 Node 설치(M1맥)

    Node는 초기부터 arm64를 지원하여 그런지 막힘없이 잘 된다.

    nvm GitHub 저장소

    nvm 설치

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.2/install.sh | bash

    nvm 버전 확인

    nvm -v

    설치 가능한 버전을 확인한다.

    nvm ls-remote

    가능하면 LTS 버전을 설치하는게 좋다. 현시점 최신 LTS는 18.12.1 이다. 18을 입력하면 알아서 Latest를 선택해준다.

    nvm install 18

    설치 확인

    nvm ls

    node 및 npm 버전 확인

    node -v && npm -v

    아키텍처 확인. arm64로 출력된다.

    node -p process.arch
