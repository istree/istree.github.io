    VuePress

    Vue.js 기반의 정적 사이트 저작 도구이다.

    전제조건

    Node 10 이상 Yarn Classic이 필요하다.

    빠른시작

    디렉토리 생성

    mkdir somedir
    cd somedir

    초기화

    yarn init

    VuePress 설치

    yarn add -D vuepress

    서브 디렉토리 생성

    mkdir docs
    echo '# 안녕' > docs/README.md

    package.json에 스크립트 추가

    {
      "scripts": {
        "docs:dev": "vuepress dev docs",
        "docs:build": "vuepress build docs"
      }
    }

    문서 사이트를 로컬 서버로 제공

    yarn docs:dev

    설정 파일

    대상 문서 디렉토리 내에 .vuepress 디렉토리를 만들고 config.js를 추가한다.

    .
    ├─ docs
    │  ├─ README.md
    │  └─ .vuepress
    │     └─ config.js
    └─ package.json
    module.exports = {
      title: '안녕 VuePress'
      description: '방금 만들었어요'
    }

    네비게이션

    사이트 상단에 네비게이션을 만든다. config.js에 다음을 추가한다.

    // .vuepress/config.js
    module.exports = {
      themeConfig: {
        nav: [
          { text: '홈', link: '/' },
          { text: '가이드', link: '/guide/' },
          { text: '구글', link: 'https://google.com' }
        ]
      }
    }

    핫 리로드

    package.json에 다음 항목 추가

      "scripts": {
        "preinstall": "npx npm-force-resolutions",
      },
      "resolutions": {
        "watchpack": "1.6.1"
      }

    npm install 후 VuePress를 실행하면 정상적으로 작동한다.
