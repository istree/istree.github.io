    Markdown

    기본 문법

    머리말

    샵(#)을 반복적으로 입력한다. H1 부터 H6까지 있다.

    # H1
    ## H2
    ### H3
    #### H4
    ##### H5
    ###### H6

    개행

    문장의 마지막에 공백을 2개 입력하면 한줄 개행
    Enter를 두번 입력하면 두줄 개행된다.

    공백 2개 입력  
    한줄 개행

    Enter 두번 입력

    두줄 개행

    굵게

    **강조**

    기울임

    *강조*

    인용

    들여쓰기로 강조한다.

    > 인용1
    >> 인용2
    >>> 인용3

    순차 목록(Ordered List)

    앞에 번호가 붙는 목록

    1. 첫째줄
    2. 둘째줄
    3. 셋째줄

    비순차 목록(Unordered List)

    앞에 점이 붙는 목록

    - 첫째줄 
    - 둘째줄 
    - 셋째줄

    코드

    `코드`

    ---
