    08 제목

    Jekyll의 경우 파일명을 기반으로 제목을 자동으로 만들어 준다.

    11ty는 title을 자동으로 만들어주지 않아서 FrontMatter에 직접 title 항목을 기재해야 한다.

    그러나 모든 글에 제목을 일일히 입력해서 관리하는 것은 쉽지 않다.

    다음과 같이 title 필터를 만들어서 쓰는 것이 좋다.

    function title(page) {
        let pageTitle = page.data ? page.data.title : undefined;
        return pageTitle ? pageTitle : removeHyphen(page.fileSlug);
    }

    title 유무를 확인하고 없다면 fileSlug로 대체한다.

    fileSlug는 파일명에서 날짜를 뺀 나머지 텍스트에 공백을 하이픈으로 대체한 것 이다.

    가령 제목이 2022-12-01-11ty-정적-웹사이트-생성기.md 라면

    fileSlug 값은 11ty-정적-웹사이트-생성기 가 된다.

    fileSlug에서 하이픈을 제거하면 Jekyll 처럼 파일명을 제목으로 사용 할 수 있다.

    위 필터를 다음과 같이 사용하면 Jekyll의 제목처럼 사용할 수 있다.

    {{ page | title }}
