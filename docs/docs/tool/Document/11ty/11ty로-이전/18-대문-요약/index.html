    18 대문 요약

    마지막으로 대문 페이지를 만들어준다.

    딱히 생각나는게 없어서 기록, 정리의 마지막으로 수정한 10개의 게시물 링크를 출력하기로 한다.

    배열을 부분적으로 자르기 위해서 slice 필터를 만든다.

    module.exports = function(config) {
        config.addFilter("slice", slice);
    }

    function slice(target) {
        var params = Array.prototype.slice.call(arguments, 1);
        return Array.prototype.slice.apply(target, params);
    }

    기록 컬렉션에 대해 마지막 10건의 게시물 링크를 출력한다면

    {% assign latest10Posts = collections['기록'] | reverse | slice : "0", "10" %}
    {% for post in latest10Posts %}
        <a href="{{ post.url | normalize }}">{{ post | defaultPageTitle }}</a>
    {% endfor %}

    다른 컬렉션도 똑같이 적용한다.
