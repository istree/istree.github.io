    12 태그별 페이지 생성

    게시물을 태그별로 분류 했다면 분류별로 목록이 있어야 확인하기 편리하다.

    게시물에 태그를 넣다보면 언제든지 새로운 분류의 태그가 추가될 수 있기 때문에 한땀한땀 만드는 것보다 태그 목록에 따라 자동으로 생성해주는 것이 좋다.

    데이터로 페이지를 생성하는 pagination 기능을 활용하여 쉽게 만들 수 있다.

    tagsEach.html 파일을 만들어 다음과 같이 FrontMatter 를 작성한다.

    ---
    title: 태그
    layout: layouts/archive.liquid
    pagination:
      data: collections
      size: 1
      alias: tag
    permalink: /tags/{{ tag }}/
    eleventyComputed: {
      "title": "{{ tag }}"
    }
    ---

    요약하자면 collections를 순회하며 /tags/태그명/index.html 페이지를 만든다.

    가령 태그가 [기록, 11ty, Jekyll] 이라면

    다음과 같이 생성된다.

    tags/
    ├─ 기록/
    │  ├─ index.html
    ├─ 11ty/
    │  ├─ index.html
    ├─ Jekyll/
    │  ├─ index.html
