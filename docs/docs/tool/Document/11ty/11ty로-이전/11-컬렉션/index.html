    11 컬렉션

    11ty에는 collections 변수가 있다.

    collections는 앞서 태그 별로 분류한 게시물의 목록이다.

    .이나 [] 연산자로 사용한다.

    앞서 예를 든 '기록' 태그의 모든 게시물의 목록에 접근할 경우

    {% assign posts = collections.기록 %}

    또는

    {% assign posts = collections['기록'] %}

    과 같이 사용한다.

    모든 게시물에 접근 하기 위해 all 이라는 특별한 태그가 있다.

    collections.all 은 모든 게시물의 목록이다.

    {% assign allPages = collections.all %}

    모든 페이지의 url을 출력한다면

    {% for post in collections.all %}
      {{ post.url }} <br/>
    {% endfor %}
