    nvm

    노드 버전 매니저. Node를 설치하는 도구이다.

    설치

    nvm 설치

    $ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash

    최신 버전의 Node 설치

    nvm install node

    최신 버전의 Node 사용

    nvm use node

    버전 확인

    node -v && npm -v

    아키텍처

    아키텍처 확인

    node -p process.arch

    아키텍처를 x64로 변경

    arch -x86_64 zsh

    아키텍처를 arm64로 변경

    arch -arm64 zsh

    버전변경

    Node 16버전 설치

    nvm install 16

    Node 16버전 사용

    nvm use 16

    설치된 Node의 목록을 확인하여 원하는 Node로 변경한다.

    nvm ls

    터미널 시작시 Node 16버전을 기본으로 사용한다.

    nvm alias default 16

    업그레이드

    nvm upgrade
