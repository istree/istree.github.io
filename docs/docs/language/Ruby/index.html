    Ruby

    Rbenv를 사용하여 Ruby 설치하기

    ruby-build tool 설치

    sudo apt update
    sudo apt install git curl libssl-dev libreadline-dev zlib1g-dev autoconf bison build-essential libyaml-dev libreadline-dev libncurses5-dev libffi-dev libgdbm-dev

    rbenv와 ruby-build 스크립트 설치

    curl -sL https://github.com/rbenv/rbenv-installer/raw/master/bin/rbenv-installer | bash -

    rbenv 경로 추가

    echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
    echo 'eval "$(rbenv init -)"' >> ~/.bashrc
    source ~/.bashrc

    Ruby 설치

    rbenv install 2.5.1
    rbenv global 2.5.1

    설치된 Ruby 버전 확인

    ruby -v

    M1맥에서 rbenv의 루비 설치가 실패할때

    M1맥에서 rbenv로 설치 시 m1 mac Inspect or clean up the working 으로 시작하는 메시지가 나오면서 make 오류가 날 때가 있다.

    다음 옵션을 붙이면 해결된다. ..* 에 버전을 넣는다.

    RUBY_CONFIGURE_OPTS=--with-readline-dir="$(abrew --prefix readline)" \rbenv install *.*.*
